# MathDeck Generator — Target Users & Institutions

## Individual Users

| Persona | Why they'd use it |
|---|---|
| University math professors | Rapid lecture prep across undergrad + grad courses |
| Teaching assistants | Generate review sessions and recitation slides fast |
| Private math tutors | Professional-looking decks for each student topic |
| PhD/masters students | Prepare for qualifying exam presentations or seminars |
| High school AP/IB teachers | Calculus, stats, linear algebra course content |
| Online course creators | Structured slide content for Udemy/Coursera recordings |
| Self-learners | Structured study material for self-paced learning |
| Actuarial exam candidates | Probability, statistics, numerical analysis decks |

## Institutions

| Institution Type | Use Case |
|---|---|
| University math departments | Department-wide subscription for faculty + TAs |
| Community colleges | Affordable lecture prep for adjunct instructors |
| Online learning platforms (Coursera, edX, Brilliant) | Content generation pipeline for math modules |
| Test prep companies (Kaplan, Princeton Review) | GRE quant, SAT math, actuarial prep materials |
| Coding bootcamps | Linear algebra, probability, optimization for ML curricula |
| Quant finance training programs | Stochastic calculus, optimization, numerical analysis |
| High school networks (IB World Schools, AP programs) | Standardized deck templates across campuses |
| Corporate L&D teams | Data science / ML math foundations training |

## Highest-Value Targets (Priority)

1. **University TAs** — high volume users, departmental budgets, strong word-of-mouth
2. **Online course creators** — power users who generate many decks, strong Pro upgrade motivation
3. **Test prep companies** — institutional deals, predictable recurring revenue
